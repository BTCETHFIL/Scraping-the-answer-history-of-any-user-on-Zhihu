### 单个 MD 文件内容结构

每个 Markdown 文件包含：

1. **标题**：知乎问题的标题，带原文链接
2. **元数据**：作者昵称、知乎主页链接、发布时间、点赞数

4. **文字内容**：可直接搜索、复制的纯文本内容（Markdown 格式）

## 八、技术说明

- **工具版本**：v2.0（知乎内容抓取工具）
- **采集方式**：基于 Playwright 的自动化浏览器，访问知乎公开页面，截图并提取文字
- **内容完整性**：每个回答均保留问题标题、回答正文、发布时间、原文链接、点赞数
- **截图保真**：截图保留原始页面排版（题目 + 回答区），不含右侧推荐栏等无关内容
- **离线可用**：所有图片已 base64 内嵌（v2.0 达到 100% 内嵌率），文件自包含，不依赖网络或额外文件
- **可搜索**：文字内容为纯文本 Markdown，支持全文搜索（Ctrl+Shift+F 在 VS Code 中可跨所有文件搜索关键词）
- **跨平台**：MD 文件为纯文本，Windows/macOS/Linux 均可直接打开查看
- **增量更新**：支持后续对新增用户或新增回答进行增量采集，不影响已有归档
- **法务证据**：工具内置法务模式（forensic_mode），每篇回答保留独立截图 + 原文链接，可追溯验证
- **采集时间**：2026 年 6 月
